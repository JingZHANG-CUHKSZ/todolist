# 📝 我的待办事项管理器

一个现代化的待办事项管理网站，支持实时添加、编辑、删除和筛选功能。

## ✨ 功能特色

- 📱 响应式设计，支持手机和电脑
- 💾 本地存储，数据不丢失
- 🎨 现代美观的界面设计
- ⚡ 丰富的交互功能
- ⌨️ 键盘快捷键支持

## 🚀 使用方法

1. 在输入框中输入待办事项
2. 点击"添加"按钮或按回车键添加
3. 点击圆形按钮标记完成状态
4. 双击文本可以编辑内容
5. 使用筛选按钮查看不同状态的任务
